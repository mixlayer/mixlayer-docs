# Quickstart

You'll have a working chat completion in three steps.

## 1. Get an API key

Sign up at [console.mixlayer.com](https://console.mixlayer.com/sign-up) and create an API key from the [API Keys](https://console.mixlayer.com/app/api-keys) page in the console. Export it in your shell:

```bash
export MIXLAYER_API_KEY="..."
```

## 2. Send your first request

Mixlayer is OpenAI-compatible. The base URL is `https://models.mixlayer.ai/v1`.

<Tabs items={['curl', 'Python']}>
<Tabs.Tab>
```bash file=quickstart.sh
curl https://models.mixlayer.ai/v1/chat/completions \
  -H "Authorization: Bearer $MIXLAYER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "qwen/qwen3.5-4b-free",
    "messages": [
      {"role": "user", "content": "Tell me a fun fact about chihuahuas."}
    ]
  }'
```
</Tabs.Tab>
<Tabs.Tab>
```python file=quickstart.py
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["MIXLAYER_API_KEY"],
    base_url="https://models.mixlayer.ai/v1",
)

response = client.chat.completions.create(
    model="qwen/qwen3.5-4b-free",
    messages=[
        {"role": "user", "content": "Tell me a fun fact about chihuahuas."},
    ],
)

print(response.choices[0].message.content)
```
</Tabs.Tab>
</Tabs>

<Callout type="info">
  `qwen/qwen3.5-4b-free` is on the free tier and great for trying
  things out. See the [Models](/models) page for the full catalog.
</Callout>
