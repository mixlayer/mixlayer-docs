# `gen(options)`

`gen` instructs the underlying model to generate text and append it to the context window. 

### Arguments

`options.len` <span className="font-mono">string</span><br/>
The length limit (in tokens) for the generated text.

---
`options.role` <span className="font-mono">string</span><br/>
For instruct models, this will emit the correct tokens to label your `input` with the correct role. This is typically `user`, `assistant` or `system` but can vary from model to model.

---
`options.regex` <span className="font-mono">string</span><ExperimentalBadge className="ml-1"/><br/>
A regular expression that will restrict the output of the generated text to the pattern specified.

---
`options.hidden` <span className="font-mono">boolean</span><br/>
If `true`, the text will not be visible in the output in the output by default.
